<h3>How to Use it?</h3>

1. open VS Code, or any of your code editor.
2. open a new terminal.
3. git clone https://github.com/sandeepk-480/Simple-Portfolio-Project.git
4. cd Simple-Portfolio-Project
5. npm i
6. npm run dev
